# FastAPI tutorial for study

- requirements
  - Python 3.10
    - `brew install python@3.10`
  - poetry
    - `curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python -`

```bash
poetry env use /opt/homebrew/opt/python@3.10/bin/python3
poetry install
poetry shell
python main.py
python homework.py
```
